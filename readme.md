# MPCP Simulation Project

Dependencies:

+ Project aimed to be ran on a LAMP or WAMP stack. 
+ `mysqli`, `openSSL` and `cURL` should be enabled for PHP
+ API uses `mod_rewrite`. A `.htaccess` file is attached.
